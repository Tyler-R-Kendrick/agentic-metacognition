# gh aw init

Initialize a repository for authoring GitHub Agentic Workflows.

## When to use it

Use in a repo that does not yet have the gh-aw authoring scaffolding.

## How to use it

1. Start with `gh aw init --help` to see the flags for your installed version.
2. Run the smallest command that answers your immediate need.
3. Review the diff or GitHub-side effect before moving on when the command changes repository state.

## Example commands

- `gh aw init`
- `gh aw init --help`

## Notes

- This may create `.github/agents/` and editor integration files; only use it intentionally.

## Verify

- Re-run `gh aw init --help` if you are unsure about supported flags in the installed release.
