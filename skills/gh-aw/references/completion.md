# gh aw completion

Generate shell-completion scripts for `gh aw` commands.

## When to use it

Use when you want tab completion in bash, zsh, fish, or another supported shell.

## How to use it

1. Start with `gh aw completion --help` to see the flags for your installed version.
2. Run the smallest command that answers your immediate need.
3. Review the diff or GitHub-side effect before moving on when the command changes repository state.

## Example commands

- `gh aw completion bash`
- `gh aw completion zsh`

## Notes

- Pipe the output into the shell-specific completion location for your environment.

## Verify

- Re-run `gh aw completion --help` if you are unsure about supported flags in the installed release.
